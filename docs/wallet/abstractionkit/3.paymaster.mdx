A `CandidePaymaster` is a class interacts with Candide's ERC-4337 Paymaster API.

The `CandidePaymaster` class sets up a Paymaster Instance with:

1. Paymaster URL `string`

2. Entrypoint address `string`

## Usage

### Import

```ts
import { CandideValidationPaymaster } from "abstractionkit";
```

### How to use

Initialize a Paymaster with your RPC url

:::info
Get an API key from the [dashboard](httsp://dashboard.candide.dev)
:::

```ts title="paymaster.ts"
import { CandidePaymaster } from "abstractionkit";

const paymasterRpc = "https://api.candide.dev/paymaster/$version/$network/$apikey";

const paymaster: CandidePaymaster = new CandidePaymaster(paymasterRPC);
```

Then you can consume Paymaster methods:

```ts
const supportedEntrypoints = await paymaster.getSupportedEntrypoint();
```
